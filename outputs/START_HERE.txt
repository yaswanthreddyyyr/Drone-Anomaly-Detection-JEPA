🏗️ PIPELINE ARCHITECTURE
═════════════════════════════════════════════════════════════════════════════

Raw Flight Logs
    ↓
PREPROCESSING: Chunks (1024 waypoints, 11 features)
    ↓
JEPA v3: Self-supervised training → 384-dim embeddings
    ├─ 3-layer encoder [1024→768→384]
    ├─ Adaptive masking curriculum (12%→72%)
    ├─ 140 epochs on A100 (84.9 minutes)
    └─ CLS pooling for chunk-level representation
    ↓
FEATURE ENGINEERING: Scaled → PCA compressed
    ├─ StandardScaler normalization

    ↓
ANOMALY DETECTION: Local Outlier Factor
    ├─ Manhattan distance metric (L1)
    ├─ k=22 neighbors for density estimation
    └─ Continuous anomaly scores
    ↓
THRESHOLD CALIBRATION: Validation-based tuning
    ├─ 260 threshold percentiles tested
    ├─ FAR cap constraint: ≤ 0.20
    └─ Selected threshold: 1.552
    ↓
FINAL PREDICTION: Binary anomaly classification
    └─ Score ≥ 1.552 → Anomaly; else → Normal

💡 KEY INSIGHTS
═════════════════════════════════════════════════════════════════════════════

1. Distance Metric Selection is Critical
   └─ Manhattan (L1) better than Euclidean (L2) for tampering detection
   └─ +14.3% improvement from metric change alone

2. JEPA Self-Supervised Learning Works
   └─ No need for anomaly labels during training
   └─ Model learns normal flight patterns
   └─ Transfers well to anomaly detection

3. Density-Based > Isolation-Based
   └─ LOF (density) outperformed IF (isolation)
   └─ Exploits local structure in learned embeddings
   └─ Better recall and F1 scores

4. PCA Sweet Spot at 124 Dimensions
   └─ Retains 99.85% variance
   └─ Reduces noise without losing signal
   └─ Better LOF distance computations

5. Validation Calibration Essential
   └─ Threshold tuning on validation set prevents overfitting
   └─ FAR constraint keeps false alarms under control
   └─ Consistent performance across test splits
